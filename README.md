# JpSki

[![Build Status](https://travis-ci.org/tsmsogn/jp_ski.svg?branch=master)](https://travis-ci.org/tsmsogn/jp_ski)

Japan ski areas

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'jp_ski'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jp_ski

## Usage

Get all ski areas

    JpSki::Ski.all
